# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release of Laravel WordPress Connector
- WordPress database integration with Eloquent models
- Complete Filament admin panel resources
- Dashboard widgets for statistics
- Wp facade for easy queries
- Support for WordPress users, posts, comments, terms, and options
- Meta data management for all entities
- Relationship mapping between WordPress entities
- Publishable configuration file
- Comprehensive documentation

### Features
- **Models**: WpUser, WpPost, WpComment, WpTerm, WpOption, WpMeta
- **Relationships**: Full relationship mapping between WordPress entities
- **Filament Resources**: Complete CRUD operations for all WordPress entities
- **Dashboard Widgets**: Recent posts, user count, and comment count widgets
- **Facade**: Easy-to-use Wp facade for complex queries
- **Configuration**: Flexible database connection configuration
- **Meta Support**: Full support for WordPress meta data
